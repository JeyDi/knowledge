Level: A1 - 5

# Personal Pronouns

* "Yo" (I)
* "tú" (you, informal)
* "usted" (you, formal)
* "él" (he), "ella" (she)
* "nosotros / nosotras" (we)
* "vosotros / vosotras" (you, informal)
* "ellos / ellas" (they)
* "ustedes" (you, formal)

# The verb "estar" / To be
* (yo) estoy
* (tú) estás
* (él / ella / usted) está
* (nosotros / nosotras) estamos
* (vosotros / vosotras) estáis
* (ellos / ellas) están
